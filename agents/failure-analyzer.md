# Failure Analyzer Agent

## Problem

Analyzing test failures manually is time-consuming and inconsistent.

## Solution

An AI agent that analyzes logs and predicts root causes.

---

## Capabilities

* Classifies failures:

  * Product bug
  * Test issue
  * Environment issue
* Suggests possible fixes
* Summarizes logs

## Example

**Input:**

```
TimeoutError: API did not respond
```

**Output:**

```

Type: Environment issue  
Suggestion: Check API response time
```
